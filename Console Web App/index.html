Visual Studio Code

Visual Studio Code-About

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

Visual Studio Code-Setup

  • Download and install VS Code.
  • Create a new file.
  • See an overview of the user interface.
  • Install support for your favorite programming language.
  • Change your keyboard shortcuts and easily migrate from other editors using keymap extensions.
  • Customize your editor with themes.
  • Explore VS Code features in the Interactive Editor Playground.

Console Screen

Console Screen-About

The “Console” Tab. The heart of any developer tool is the “console” tab where you can output debug and execute commands in to the current web page.

Console Screen-How To View

To open the console in Chrome, use this keyboard shortcut: Cmd + Option + J (on a Mac) or Cmd +Shift +J (on Windows). As an alternative, you could right-click on the webpage and click "Inspect," and the developers window will appear.

Console Screen-How To Print

Example: "console.log("Hello world!");"
